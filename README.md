# Student Database Management System

A simple relational database project built with MySQL to manage student information, courses, enrollments, and calculate GPAs. This project demonstrates core SQL concepts including table creation, data manipulation, and complex queries using joins and aggregate functions.

## 📚 About The Project

This database system is designed to help educational institutions track:

- Student personal information and enrollment
- Course catalog with credit hours
- Student course registrations and grades
- GPA calculation for academic reporting

The project simulates a real-world academic database that could power a student portal or grade management system.

## 🗄️ Database Schema

The database consists of the following tables:

### Tables

| Table | Description |
| --- | --- |
| `students` | Stores student details (ID, name, email, enrollment date) |
| `courses` | Contains course information (course name, credits) |
| `enrollments` | Junction table linking students to courses with grades and semester |
